#adminlte-vue 使用方法

> 这是adminlte和vue结合的一个后台管理模板,里面只有几个简单的架子,希望可以给大家起到一个抛砖引玉的作用.

## 安装方法
1.本地安装nodejs和vue-cli
2.下载项目到本地
3.开始安装

``` bash
npm install
```
4.开发模式
``` bash
npm run dev
```
5.打包发布
``` bash
npm run build
```
